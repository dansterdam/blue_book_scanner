11. BRIEF SUMMARY AND ANALYSIS
Observers noted brilliant color object about the size of a match head traveling to the NE. One observer said object had same intensity in brightness as aircraft. The object disappeared behind a building was reported in both cases. One observer reported observance of a red light.

Description is consistent with that of an aircraft.
